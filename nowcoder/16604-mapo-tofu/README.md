# 牛客 16604 - 概率异或子集计数

## 题目链接

- 来源：[牛客网](https://ac.nowcoder.com/acm/problem/16604)
- 题意：给定 `n` 枚硬币，第 `i` 枚硬币正面朝上的概率为 `p_i`。选择一个子集 `S`，把子集中的硬币各抛一次，正面记为 `1`，反面记为 `0`，将所有结果异或得到 `Y`。问有多少个子集满足 `P(Y=0)=P(Y=1)`。

## 思路

因为 `Y` 只有 `0` 和 `1` 两种取值，所以：

```text
P(Y=0) = P(Y=1)
```

等价于：

```text
P(Y=0) - P(Y=1) = 0
```

注意：

```text
(-1)^Y =  1, Y = 0
(-1)^Y = -1, Y = 1
```

因此：

```text
E[(-1)^Y] = P(Y=0) - P(Y=1)
```

又因为异或满足：

```text
(-1)^(X1 xor X2 xor ... xor Xm)
= (-1)^X1 * (-1)^X2 * ... * (-1)^Xm
```

对子集 `S` 有：

```text
E[(-1)^Y]
= E[product((-1)^Xi)]
= product(E[(-1)^Xi])
```

单个硬币：

```text
E[(-1)^Xi]
= P(Xi=0) * 1 + P(Xi=1) * (-1)
= (1 - p_i) - p_i
= 1 - 2p_i
```

所以：

```text
P(Y=0) - P(Y=1) = product(1 - 2p_i), i in S
```

要让左右概率相等，就需要：

```text
product(1 - 2p_i) = 0
```

一个乘积为 `0`，当且仅当至少有一项为 `0`，即：

```text
1 - 2p_i = 0
p_i = 0.5
```

所以，满足条件的子集必须至少包含一枚概率为 `0.500000` 的公平硬币。

设公平硬币数量为 `k`：

- 所有子集数量为 `2^n`
- 不包含任何公平硬币的子集数量为 `2^(n-k)`
- 满足条件的子集数量为 `2^n - 2^(n-k)`

这也自动排除了空集：空集不包含公平硬币，异或结果恒为 `0`，不满足 `P(Y=0)=P(Y=1)`。

## 复杂度

- 时间复杂度：`O(n)`，每组数据只需要统计 `0.500000` 的数量。
- 空间复杂度：`O(1)`，除输入存储外只使用常数额外空间。

## 代码

```python
import sys
a = sys.stdin.read().split()
t = int(a[0])
i = 1
ans = []
for _ in range(t):
    n = int(a[i])
    i += 1
    k = a[i:i+n].count("0.500000")
    i += n
    ans.append(str((1 << n) - (1 << (n - k))))
print("\n".join(ans))
```
